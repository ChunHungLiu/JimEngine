Engine class (Engine.h)

The Engine class is the class which basically runs the game. Each frame is updated through this class, and all major systems are created and destroyed in this class.

The engine can be accessed through the "engine" global variable at any time, although is mostly used for statistics and information.

The only reason one would want to create a class derived from Engine is if they want to create a custom window setup or if they want to alter the way the frame update works, but such things are rarely needed.

Engine protected variables

BOOL bCustomWindow;

If you are implementing a custom window, create your window and then set this to TRUE before calling Super::Init() in your Init().

Engine class functions

class BASE_EXPORT Engine : public Object
inline DWORD GetCurFPS() const

Returns the current FPS.

inline DWORD NumLights() const

Returns the number of visible dynamic lights in the scene.

inline DWORD NumPolys() const

Returns the number of polgons being drawn per frame.

inline DWORD NumVisibleAreas() const

Returns the number of visible areas (PVSs for indoor, Quadtree areas for outdoor and tiled levels)

BOOL InEditor() const

Returns TRUE if currently in the editor, otherwise FALSE.

virtual void Update(BOOL bProcessTime=TRUE);

This is the main frame update function. This is called automatically unless you have a custom update loop, in which you can call it as you please. This function can be implemented but make absolutely sure to call Super::Update(bProcessTime).
