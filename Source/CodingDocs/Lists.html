Lists (Template.h)

I implemented my own list template classes, hope no one minds. Relatively straight-forward to use.

List<T> class

template<typename T> class List
inline T* Array() const
inline unsigned int Num() const
inline unsigned int Add(const T& val)

unsigned int SafeAdd(const T& val)

Adds only if there is no duplicate value

inline void Insert(const T& val, unsigned int index)

inline void Remove(unsigned int index)
inline void RemoveItem(const T& obj)

inline void CopyArray(const T *new_array, unsigned int n)
inline void AppendArray(const T *new_array, unsigned int n)

inline void CopyFrom(const List<T>& list)
inline void AppendList(const List<T>& list)

inline void SetSize(unsigned int n)

inline void RemoveDupes()

inline void FreeAll()

inline T* CreateNew()
inline T* InsertNew(int index)

Returns the pointer offset in the list of the new list items. Despite the misleading name this does not use the new operator or anything, the function is simply for returning a pointer to the new item created.

inline List<T>& operator<<(const T& val)

Same essential functionality as Add, though returns the list itself for stacking multiple << operators.

inline T& GetElement(unsigned int index)
inline T& operator[](unsigned int index)
inline T& operator[](unsigned int index) const
inline T* operator+(unsigned int index)

inline BOOL HasValue(const T& val) const
inline DWORD FindValueIndex(const T& val) const
inline DWORD FindNextValueIndex(const T& val, DWORD lastIndex=INVALID) const

inline T& Last() const

Gets the last item in the list.

friend Serializer& operator<<(Serializer &s, List<T> &list)

SafeList<T> class

Index-safe list, unlike with regular lists. Not really used very often. Functions pretty much exactly like List otherwise

template<typename T> class SafeList : public List<T>

StringList class (XenString.h)

Use this class for safe handling of string lists.

class BASE_EXPORT StringList : public List<String>
unsigned int SafeAddI(const String& val)
inline BOOL HasValueI(const String& val) const
inline DWORD FindValueIndexI(const String& val) const
inline DWORD FindNextValueIndexI(const String& val, DWORD lastIndex=INVALID) const

Includes case-insensitive versions of these functions
