Game class (Game.h)

This class is used for the current game mode. For example, a multiplayer game would typically have multiple game modes, such as CTF, deathmatch, and such, in which you could create a version of this class for each.

Through this class you implement game-specific things, primarily your viewport and other game-mode specific things.

The current game is automatically assigned stored in the global "CurrentGame" variable in the Game constructor when one is created. It is automatically set to NULL in the destructor when the current game is destroyed.

Game protected variables

float GameSpeed;

Stores the current game speed, 1.0 is normal, below 1.0 is slower, and above 1.0 is faster.

Viewport *mainViewport;

Stores your main viewport. Create your custom viewport and assign it to this variable in your Init().

Game functions

class BASE_EXPORT Game : public Object
inline void SetGameSpeed(float speed)
inline float GetGameSpeed() const

Gets or sets the speed of the current game.

inline Viewport* GetMainViewport() const

Gets the main viewport currently being used in the current game mode.
