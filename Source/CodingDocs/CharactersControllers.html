Characters and Controllers (Character.h)

The Character class is the primary Entity class in which enemies, players, NPS, and such are manipulated by using a Controller. A character class is not inherently controlled by a player or the PC. That value is set by using a specific controller with each character in the game. This allows you to control which characters can or can't be controllable by the system or player at any given time.

Controller class functions

class BASE_EXPORT Controller : public FrameObject
inline int NumCharacters() const
inline Character* GetCharacter(int id) const

Gets a character from the list of characters that are assigned to this controller.

Controller implementable functions

virtual void GotControl()       {}
virtual void LosingControl()    {}

These two functions are called when your controller is assigned to or unassigned from a character

Character class functions

class BASE_EXPORT Character : public AnimatedEntity
inline Controller* GetController() const
Controller* SetController(Controller* controllerIn)

Gets or sets the current controller assigned to this character.
