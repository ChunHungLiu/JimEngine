Files (XFile.h)

For files, rather than use the typical functions, I use the XFile class. ...Trying.. to.. resist.. pop culture.. reference...

Global file functions

inline String GetPathFileName(CSTR lpPath)
String GetPathDirectory(CSTR lpPath)
String GetPathExtension(CSTR lpPath)

These functions will extract a file name/extension/directory from a given path

BASE_EXPORT void   XENAPI OSDeleteFile(TSTR lpFile);

XFile class

class BASE_EXPORT XFile
XFile()
XFile(CTSTR lpFile, DWORD dwAccess, DWORD dwCreationDisposition);

~XFile()

BOOL Open(CTSTR lpFile, DWORD dwAccess, DWORD dwCreationDisposition);
void Close();

dwAccess can be a combination of XFILE_READ and XFILE_WRITE.

dwCreationDisposition can be one of the four following values:

XFILE_CREATENEW
XFILE_CREATEALWAYS
XFILE_OPENEXISTING
XFILE_OPENALWAYS

BOOL IsOpen()

DWORD Read(LPVOID lpBuffer, DWORD dwBytes);
DWORD Write(LPVOID lpBuffer, DWORD dwBytes);

DWORD WriteAsUTF8(TSTR lpBuffer, DWORD dwElements);

Writes a string as UTF8.

BOOL SetFileSize(DWORD dwSize);
DWORD GetFileSize();

DWORD SetPos(int iPos, DWORD dwMoveMethod);
inline DWORD GetPos()

dwMoveMethod can be one of the three following values:

XFILE_BEGIN
XFILE_CURPOS
XFILE_END
