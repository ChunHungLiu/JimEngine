Music Manager (Music.h)

This class is used for playing music with ogg files. Yes, I added in all the bug fixes from TNM to this one ^_^

This class is automatically created in Engine::Init() to the global variable MM.

MusicManager class functions

class BASE_EXPORT MusicManager : public FrameObject
BOOL Play(CTSTR lpFileIntro,
          CTSTR lpFileLoop,
          float playPos=0.0f,
          MusicTransition transitionType=MusicTransition_Instant);

Plays a song with at playPos. lpFileIntro may be NULL if you don't have an intro.

transitionType can be one of the following types of this enum:

enum MusicTransition
{
    MusicTransition_Instant,
    MusicTransition_FastFade,  //1 second
    MusicTransition_Fade,      //2 seconds
    MusicTransition_SlowFade   //3 seconds
};

void Stop();
float GetCurrentPos();
