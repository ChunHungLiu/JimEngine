Cameras and Viewports (Camera.h)

A viewport is a regular 2D window object in which the current 3D level is drawn.

Cameras are assigned to a viewport window which are used as a means of viewing the 3D world.

For main game viewports, one typically takes keyboard/mouse input through this window as well.

Camera class functions

class BASE_EXPORT Camera : public Entity
void SetPerspective(float fovy, float aspect, float znearIn, float zfarIn);
void SetFrustum(float leftIn, float rightIn, float topIn, float bottomIn, float znearIn, float zfarIn);
void SetOrtho(float leftIn, float rightIn, float topIn, float bottomIn, float znearIn, float zfarIn);

Sets the camera view information. This information will be automatically loaded as a projection matrix when the cmaera is used for rendering a level.

inline ViewClip GetClip() const

Gets the current clipping planes for the camera. These clipping planes (through the ViewClip structure) are used for discerning visibility of objects and areas through the camera. You should almost never need to use this because visibility is automatically handled.

inline Viewport *GetAssignedViewport()

If the camera is assigned to a viewport, returns the viewport, otherwise NULL.

inline BOOL IsSoundCamera()

Returns TRUE if this camera can be used to hear sound with, otherwise FALSE.

inline void SetSoundCamera(BOOL bPlaySound)

Sets whether this camera can relay sound through it or not.

inline BOOL IsPerspective()

Returns TRUE if this camera is a perspective/frustrum camera, otherwise FALSE if it's an orthographic camera.

inline float Left() const
inline float Right() const
inline float Top() const
inline float Bottom() const
inline float Near() const
inline float Far() const

Returns the main projection values.

Viewport class functions

When this window is rendered, will draw the world through the current camera assigned to it.

class BASE_EXPORT Viewport : public ControlWindow
void    SetCamera(Camera *cam);
Camera *GetCamera();

Sets or retrieves the current camera assigned to this viewport.
